Detailed Description

Extended test context for for BPF_PROG_TYPE_CGROUP_SOCK_ADDR program type. This context can be used for testing eBPF programs that call bpf_sock_addr_get_network_context helper function. eBPF programs that do not call bpf_sock_addr_get_network_context helper function, can use bpf_sock_addr_t context.
